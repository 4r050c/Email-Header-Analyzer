Settings

MxToolbox API

Stored only in this browser's localStorage. Sent only in the Authorization header of requests to the hosts below — never anywhere else. When a key hits its daily quota, the tool automatically retries with the next one.

MxToolbox's API does not publish CORS headers for browser use. If direct requests fail, deploy a small relay (a one-file Cloudflare Worker or similar) that forwards to api.mxtoolbox.com and adds Access-Control-Allow-Origin, then point this field at it. Leave blank to call MxToolbox directly.

IP Geolocation (free, keyless, separate service)

Not a MxToolbox feature. Sends each public IP found in the header chain to ipwho.is over HTTPS. Turn off if you'd rather not share hop IPs with a third party.

Data & Privacy

Everything runs in your browser. Header text is never uploaded anywhere. The only network calls this tool makes are the ones you enable above, directly from your browser to MxToolbox / ipwho.is / your configured proxy.

Runs entirely client-side. Not affiliated with or endorsed by MxToolbox. DNS/blacklist data, when enabled, comes from the MxToolbox API using your own API key.
